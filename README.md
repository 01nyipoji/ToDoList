To-Do List App

A simple and elegant To-Do List built with React

What It Does
Bible Verse Generator: Random bible verse generator to keep user motivated and on track
Add Tasks:Quickly jot down tasks you want to complete.
Delete Tasks: Remove tasks you no longer need.
Move Tasks: Move task up and down according to order of priority.
Clean UI: Simple, responsive, and easy on the eyes.


Tech Stack
React JS
CSS for styling
Git & GitHub (SSH) for version control and hosting

 How I Set It Up
1. Created a local React project (`npx create-react-app to-do-list`)
2. Initialized Git (`git init`)
3. Linked it to GitHub using SSH
